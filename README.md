# PuppyAppServer

這是提供給國立成功大學統計學系所使用的調查APP  
主要提供功能是使用APP收集遊蕩犬(流浪狗)的照片及相關數據  
收集的數據包括：GPS位置追蹤、照片、照片時間及位置、調查用相關表單數據  
 
此APP目前有在APP Store上架，最後更新日期為：2020/08/26
